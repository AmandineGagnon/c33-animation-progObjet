Faire disparaitre un élément

  1. Dans le fichier JavaScript externe, faire en sorte de détecter lorsque l'usager clique sur le noeud main.
  2. Lorsque l'usager clique sur ce noeud (main), faire une animation permettant de le faire disparaitre avec le temps. Utilisez un setTimeout pour y parvenir.
  3. Ajouter le code permettant d'arrêter l'animation lorsque l'opacité est <= à zéro.
